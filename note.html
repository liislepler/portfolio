Note

Frontend Development / UI/UX Design

Vue.js / JavaScript / CSS / REST API / SQLite / Express.js / Node.js / GitHub

Note is a collaborative platform for students to share insights related to various courses through note-taking. The frontend of the application was developed using the Vue.js framework, connected to the with REST API and an SQLite database. Additionally, the server-side architecture was structured with Node.js using Express.js as the framework.

The application includes signup and login functions and each user possesses CRUD capabilities, allowing them to create new notes and courses, read their own and others' notes, update and edit their notes, and delete content if needed.

Throughout this project, I learned a lot about Client-Server Communication and code structuring as well as code sharing.
