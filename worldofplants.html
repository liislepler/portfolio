The World of Plants

Frontend Development / UI/UX Design / Graphic Design / Illustration

JavaScript / HTML / CSS / three.js / p5.js / anime.js / GitHub / Illustrator

I was part of a game developing project using JavaScript and JavaScript libraries. Inspired by classics like Snakes & Ladders, The Circus Game, and Animal Crossing, my team aimed to create a digital board game. Prior to this project, I had grasped the fundamentals of JavaScript and had experimented with smaller games. However, this project pushed me to expand my programming skills significantly.

During the development process, I delved into Three.js and explored the creation of 3D animated objects for the first time.
