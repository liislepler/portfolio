Salerno

UI/UX Design / Frontend Development

HTML / SCSS / CSS / JavaScript / GitHub / Figma

Salerno was done as a group project where my team developed a website using a combination of HTML, CSS/SCSS, and JavaScript. This project was our introduction to using GitHub for code sharing, emphasizing the significance of clear coding practices and collaborative version control. The resulting webpage has a responsive design with viewing experiences across various devices, including phones, tablets, and desktops. Through this project, I gained valuable insights into code collaboration.

Co-creators: Elin Tri, Napsugár Takács, Divine Brisibe
