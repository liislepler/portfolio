Luca Deli

UI/UX Design / Frontend Development

Figma / PHP / WordPress / CSS / JavaScript / Photoshop

I created Luca Deli, a WordPress project, as part of a work test. With a timeframe of 15 hours, I designed and developed a homepage for Luca Deli, an online food ordering platform specializing in vegetarian charcuterie boards. Starting with the initial design on Figma, I converted it into a responsive WordPress site, developing a custom theme using PHP, CSS, and JavaScript.

This small project taught me a lot about working within tight timeframes while also ensuring the delivery of a functional website with user-friendly UI/UX design. It allowed me to strategize on differentiating the platform from competitors in the online meal ordering landscape.

The website features a homepage, an ordering page allowing users to customize the amount and size of their chosen board, a client contact information page, and an order confirmation page.
